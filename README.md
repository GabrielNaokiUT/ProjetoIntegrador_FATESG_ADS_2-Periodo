# ProjetoIntegrador_FATESG_ADS_2-Periodo
Projeto Integrador SENAI-Fatesg 2025/2 ADS 2ºPeríodo: GYNLOG - Sistema de Gestão de Frota

Software desktop desenvolvido em Java para gerenciamento de despesas de veículos da transportadora GynLog. O sistema permite o cadastro de frotas, categorização de gastos e geração de relatórios financeiros detalhados (combustível, IPVA, multas, etc.). Possui interface gráfica e persistência de dados em arquivos de texto compatíveis com planilhas eletrônicas.
